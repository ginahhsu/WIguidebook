Basic tools

Pen

Pathfinder

Illustrator has many different tools to create beautiful graphics. While graphics may be a lot more time consuming to create over photographs, the end result allows for some timeless pieces.

Pen tool:

  1. On the left-hand side, select your pen tool.
  2. Set your stroke weight, colour and fill
  3. Drawing a straight line:.
    Click and release to create an anchor point. Click on another point on the artboard to set another anchor point. Click on esc to stop drawing.
  4. Adding curves:
    Instead of clicking and releasing, click and drag to curve a line.
  5. Drawing a closed shape:
    Keep creating anchor points until you reach your first anchor point. A circle will show up next to your cursor at an end point.
  6. Use the direct selection tool to move your points and create curves.
Things to note
  • Similar tools are grouped together. Click down on the pen tool to access the different ones or click + to add anchor points, - to get rid of anchor points, and shift + c to use the anchor tool.

Paintbrush tool:

  1. On the left-hand side, select your paintbrush tool.
  2. Set your stroke weight, colour and fill
  3. Click and hold to draw.
  4. Use the direct selection tool to move your points and create curves.
Things to note
  • The paintbrush tool works very similarly to the pen tool. It will automatically smooths out lines and after, you can select anchors just as you would the pen tool.

Pathfinder tool:

  1. Create the objects you want to combine
  2. Under window in the top toolbar, select pathfinder.
  3. Select any two object to combine them.
    Unite add areas of selected components with the geometry.
    Minus Front: cut shape of a component from the geometry behind.
    Intersect: clip the geometry similar to a mask tool.
    Exclude: invert the geometry.
    Divide: separate component-filled faces from a piece of artwork.
    Trim: remove part of a hidden filled object.
    Merge: remove parts of hidden filled objects.
    Crop: break down a piece of artwork into component-filled faces.
    Outline: divide an object into edges or line segments.
    Minus Back: cut shape of a component from the geometry in front.
Things to note
  • The pathfinder tool is very useful when creating graphics. Use the Adobe help site to learn more about it
